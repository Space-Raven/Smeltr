# Active audits

Place **in-progress** Trail of Bits–style audit reports here.

**Filename:** `Agent(<job>)-audit<N>-TokenPlatform.md`  
Example: `Agent(security)-audit1-TokenPlatform.md`

**Rules (see agent charter):**

1. While a file exists here, **fix all listed problems** (or get operator deferral in the file).
2. When fully addressed: summarize in [`PROJECT.md`](../PROJECT.md) → **delete** this audit file.

Do not use this folder for long-term storage.
